# Email Sender

## Configuration

- Create a load_env.sh file and paste the following code:

```bash
export EMAIL_PASSWORD="" # Your email app password
export EMAIL_SENDER="" # Your email address
```

- Run the following command:

```bash
source load_env.sh
```

## Usage

- Run the following command:

```bash
python3 email_sender.py
```
